Main menu

The Main menu is build from floated list elements and easily extendable.

Editing the Main menu
U can easily edit the Main menu item by replacing or editing the text between the <a> element. Editing the navigation will extend the bar to the left, because they is positioned to the right.

Setting links
The navigation between different page sections is simple. To set a link to the page edit the <a href="#section_id">Example</a>


Header

The header is a block with main menu, logo and some useful blocks


Content

The Content section contains the whole content with pictures, text and plugins. The heigth of the Content depends on the height of the content inside. For customization see detail on each pages.


Footer

The footer is a block with a copyright and some useful blocks

C) JavaScript - top


The theme makes use of various JavaScripts and the jQuery framework to add dynamic functionality. Some of the scripts used include:

1. customscrollbar
2. flexslider
3. entypo
4. prettyphoto
5. scrollto
6. jQuery Tools
7. mediaelement
